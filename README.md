# Pi-WonjuYUL
Pi-WonjuYUL-A modern dashboard web application for monitoring and controlling Raspberry Pi-based IoT devices. Features real-time data visualization, remote control, and an intuitive UI. Built with Next.js, TypeScript, and React.
